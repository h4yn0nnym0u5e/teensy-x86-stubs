# teensy-x86-stubs
stub classes to allow certain teensy code to be compiled and run on a desktop PC

install:
```
git clone https://github.com/newdigate/teensy-x86-stubs.git
cd teensy-x86-stubs
mkdir cmake-build-debug
cd cmake-build-debug
cmake ..
sudo make install
```

uninstall:
```
sudo make uninstall
```
